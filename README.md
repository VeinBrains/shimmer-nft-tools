
# Shimmer NFT Tools

```
npm install
npm run dist
npm start
```

## Errors

### 1. Module in `@iota/pow-neon.js` not found.

Error:

```
Error: Cannot find module './native/_index_.node'
```
Fix: 
Need rename a file into `@iota/pow-neon.js/dist/native/index.node` into `@iota/pow-neon.js/dist/native/_index_.node`
